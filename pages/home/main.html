Click again to hide the sidebar, giving you more space to browse comfortably.

Each mod is displayed on a card with three buttons. One of them—your best friend—is the Guide button.
Clicking it will reveal a short manual explaining how to use the mod or code snippet.

How to host a Starblast mod?

Before getting started, you need a valid ECP to host a modded or private game in Starblast.
If you don’t have one, you won’t be able to host a modded game. You can purchase an ECP directly from the Starblast website.
Note: The Steam version no longer includes an ECP key, so you won’t be able to host or code modded games using it.

To begin, open a new Starblast modding tab in your preferred web browser.
Before pasting the mod into the code section, make sure the field is empty.

Once cleared, paste your chosen mod into the editor.

Press Start and enjoy!

Essential Modding Information

You can change the hosting region using the following command: 

open                       open mod game link in new tab.
clear                      clear the console.
start                      launch modded game.
stop                       kill modded game.

region 'region'              change server region.
    ex: region Europe
anything JavaScript           execute JavaScript code.
    ex: game.addAlien()
help                       display this help.

There are three available regions: Europe, America, and Asia.
