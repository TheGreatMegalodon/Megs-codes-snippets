Hosting a Mod

To host a modded game, you’ll need an ECP key (buy it on the Starblast site—Steam no longer includes one). Then, open a new Starblast modding tab, clear the editor, paste the code, and hit Start.

Quick Tip

You can switch hosting regions with a simple command. 

open                       open mod game link in new tab.
clear                      clear the console.
start                      launch modded game.
stop                       kill modded game.

region 'region'              change server region.
    ex: region Europe
anything JavaScript           execute JavaScript code.
    ex: game.addAlien()
help                       display this help.

There are three available regions: Europe, America, and Asia.
