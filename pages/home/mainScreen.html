How to host a starblast mod?

Before we start, you must have a working ECP to host a modded/private game in Starblast.
If you don’t own one, you won’t be able to host a Starblast modded/private game, or you can buy one via the starblast website directly (note: the steam app doesn't come with an ECP key anymore, therefore you won't be able to host/code modded games).

To start, you shall open a new Starblast modding tab on your favorite web browser.
Before pasting the mod into the code section, make sure it's empty.

Once you have cleared the code part, paste your favorite mod in.

Press start and enjoy !

Modding must know:

You can change the region of hosting using the following command:

Region Europe

With 3 possible regions: Europe | Amercia | Asia.
