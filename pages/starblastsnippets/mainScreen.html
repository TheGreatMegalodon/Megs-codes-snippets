Code that allows used to put a spot on the minimap at the same place of one object in-game.
Everything is automatic.

Your object needs a shape, in order for the code to work completely.
Your objects must be created and placed before you start your mod, any object that will get created after the initialization of the code, won't add a spot of the radar.

Add this line in your existing this.tick block

radarBackground.update(game);

Add this line at the very bottom of your mod.

radarBackground.create();

This code is used to add some sort of collectible spawning areas like in battle royale, its can be very useful if you plan to create a mod like battle royale.

This line needs to be added in the this.tick block in order for the code to work.

collectibleZone.manage(game, 600, 4);

"600"
Redivsent the interval in which one zone will spawn somewhere in the map.
If you want to add more areas, just add the same instruction bellow.

"4"
Redivsent the amount of collectible that will spawn inside of the zone.

This code is used to add a timer inside of a mod.

It must be updated every seconds.

This line has to be added in the this.tick block in order for the code to work.
note: That only one function this.tick can be defined in a game.

timer.update(game, "name", "color");

- name can be "anything". (keep the word short)
- color can be used in any format (rgb, hex, hsla...)

This line can be exectuted anywhere, simply "game" must be defined.

timer.remover(game, reset);

This code is used to remove the timer UI if needed.
You can also choose to reset the timer or not when removing the clock.
    reset: true or false.

Usage example:

this.tick = function(game) { if (game.step % 60 === 0) { timer.update(game, "game timer", "rgb(255, 255, 255)"); } }

This code allows to get the correct instructors for a team mode "color wise".

All you need is a "hue" component in the this.option part in order for the code to work.
A list called instructors will be created containing the instructors that works with the .team field.

Usage example:

for (let ship of game.ships) { const instructor = instructors[ship.team]; ship.instructorSays("Here is your report, Commander", instructor); }
