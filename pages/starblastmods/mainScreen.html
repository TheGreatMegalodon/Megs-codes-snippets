This mod was created to allow users to test their ships easily with a clear, customizable interface.

Simply paste all of your code below the delays block,
and fill the ships array with each variable name for the ships.

WARNING, you have to make sure to always have a "tier 1" in order
for the mode to work properly

As the example bellow

var Oresta_101 = '...'; var Scearp_201 = '...'; var Urest_202 = '...'; //(more ships...) var ships = [ Oresta_101, Scearp_201, Urest_202, //(more ships...) ];
